## Commit Message Convention

```txt
###################
# <emoji> : <description>
#
# [optional body]
#
# [optional footer(s)]
###################
# emoji 는 하단 참조
#
# body 는 더 자세한 설명
#
# footer 에 관련 이슈 설명
###################
# 예시
# ✨: 버튼 컴포넌트 추가
#
# 새 버튼 기능 구현 및 css 적용
#

```
